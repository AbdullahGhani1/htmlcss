Understanding Links in HTML

Links are created using the <a> (anchor) tag in HTML. They allow users to navigate to other pages, sections within a page, or external websites.

Basic Syntax

The anchor tag requires an href attribute, which holds the destination URL: 

<a href="https://example.com">Visit Example</a>

Services

Absolute URL
A full URL that links to an external site (e.g., https://example.com).
Relative URL
Links to a page within the same website (e.g., /about.html).
Email Link
Uses mailto: to open the user's email client (e.g., <a href="mailto:info@example.com">).

Best Practices

  • Use descriptive link text (avoid “click here”).
  • Add target="_blank" for links that open in a new tab.
  • Use rel="noopener noreferrer" for security when using target="_blank".
